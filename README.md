# Scripts for Image/Videos Processing (cvutils)

# How to Use THIS-PACKAGE
1. Add the path to this package (e.g. `<your-path>/cvutils/`) to `PYTHONPATH` 
    - `export PYTHONPATH=$PYTHONPATH:<your-path>/cvutils/` 

# Requirements
* transforms3d
* opencv 3.4.2
* python 3.6
